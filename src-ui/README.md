# src-ui

## Install dependencies

```bash
bun install
```

## Start the development server

```bash
bun dev
```

## Build the front end

```bash
bun build
```

## Format source code

```bash
bun fmt
```

## Lint code

```bash
bun lint
```

This project was created using `bun init` in bun v1.1.8.
[Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
