# build-scripts

## Install dependencies

```bash
bun install
```

## Run a script directly

```bash
bunx script.ts
```

## Run tests

```bash
bun test
```

## Format source code

```bash
bun fmt
```

## Lint code

```bash
bun lint
```

This project was created using `bun init` in bun v1.1.8.
[Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
